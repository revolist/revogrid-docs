## What this demo shows

This example shows the Angular Module integration path for RevoGrid:

- `RevoGrid` module import and template usage in a classic Angular module app.
- Basic `source` + `columns` setup for first render.
- Foundation setup you can extend with Pro/Enterprise plugins.

## When to use this setup

Use this variant when your Angular app is still module-based and you are not ready to migrate to fully standalone components.
