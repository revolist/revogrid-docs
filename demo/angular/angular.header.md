## What this demo shows

This demo focuses on richer column header UX:

- Custom header rendering for contextual labeling.
- Header-level tooltip behavior for dense column sets.
- Patterns you can reuse for status hints, validation, and metadata previews.

## Implementation notes

Use a custom header template/component when column names alone are not enough. Keep tooltip content short and actionable for faster scanning.
