## What this demo shows

This demo demonstrates custom date editing in Angular:

- Date editor wiring through column configuration.
- Controlled parsing/formatting behavior at edit boundaries.
- A reusable pattern for domain-specific editors (dates, ranges, time, currency).

## Extension ideas

Apply the same integration style for custom validators, locale-specific formatting, and keyboard-first edit UX.
